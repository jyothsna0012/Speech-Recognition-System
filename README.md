
# Speech Recognition System

A basic **speech-to-text system** using Python and pre-trained models.

## Features
- Converts short WAV audio clips to text
- Uses SpeechRecognition library (Google API)
- Error handling for unknown audio and API issues
- Works in Google Colab

## Usage
1. Upload your WAV audio file
2. Run `speech_to_text()` function
3. See transcription in output

## Technologies
- Python
- SpeechRecognition
- Google Colab
